数组的遍历方法

2021-12-17 PV:
方法 是否改变原数组 特点
forEach() 不改变 没有返回值。
map() 不改变 返回一个数组。
filter() 不改变 过滤数组。
for…of 不改 返回一个数组。
every（） 不改变 返回一个布尔值。只要有一个false,就返回false。
some() 不改变 返回一个布尔值。只要有一个true,就返回true。
find() 不改变 返回第一个符合条件的值。
findIndex() 不改变 返回第一个符合条件的值的索引值。
reduce() 不改变 返回执行之后的值。对数组正序操作。
reduceRight() 不改变 返回执行之后的值。对数组逆序操作。

forEach()

forEach() 方法对数组的每个元素执行一次给定的函数。

map（）

map() 方法创建一个新数组，其结果是该数组中的每个元素是调用一次提供的函数后的返回值。

使用案例技巧

parseInt(string, radix) 解析一个字符串，并返回一个整数或者NaN。

radix可选。表示要解析的数字的基数。该值介于2~362之间。

如果省略该参数或者其值为0，则以10为基础来解析。

如果“0x”或”0X”开头，则以16为基数。

如果该参数小于2或者大于36，则parseInt()将返回NaN。

Array.prototype.map 传递3个参数:

  • the element
  • the index
  • the array

filter()

filter() 方法创建一个新数组, 返回原数组中符合函数条件的所有元素。

for…of

for...of语句遍历可迭代对象（包括 Array，Map，Set，String，TypedArray，arguments 对象等等）定义要迭代的数据。

for...of与for...in的区别

无论是for...in还是for...of语句都是迭代一些东西。它们之间的主要区别在于它们的迭代方式。

for...in 语句以任意顺序迭代对象的可枚举属性。

for...of 语句遍历可迭代对象定义要迭代的数据。

every()

**every() **方法测试一个数组内的所有元素是否都能通过函数。

注意：若收到一个空数组，此方法在一切情况下都会返回 true。

some()

some() 方法测试数组中是不是至少有1个元素通过了函数。(在满足特定条件时停止遍历)

注意：如果用一个空数组进行测试，在任何情况下它返回的都是false。

find()

find() 方法返回数组中满足函数的第一个元素的值。

const found = array1.find(element => element > 10);

console.log(found);
// expected output: 12

findIndex()

findIndex()方法返回数组中满足函数的第一个元素的索引。若没有找到对应元素则返回-1。

reduce()

reduce() 方法对数组中的每个元素执行reducer函数(升序执行)，将其结果汇总为单个返回值。

reducer 函数接收4个参数:

  1. Accumulator (acc) (累计器)
  2. Current Value (cur) (当前值)
  3. Current Index (idx) (当前索引)
  4. Source Array (src) (源数组)

callback 被调用四次，每次调用的参数和返回值如下表：

callback accumulator currentValue currentIndex array return value
first call 0 1 1 [0, 1, 2, 3, 4] 1
second call 1 2 2 [0, 1, 2, 3, 4] 3
third call 3 3 3 [0, 1, 2, 3, 4] 6
fourth call 6 4 4 [0, 1, 2, 3, 4] 10

reduceRight()

reduceRight() 方法接受一个函数作为累加器（accumulator）和数组的每个值（从右到左）将其减少为单个值。

一共会调用四次回调函数，每次调用的参数及返回值如下：

callback previousValue currentValue index array 返回值
第一次调用 4 3 3 [0,1,2,3,4] 7
第二次调用 7 2 2 [0,1,2,3,4] 9
第三次调用 9 1 1 [0,1,2,3,4] 10
第四次调用 10 0 0 [0,1,2,3,4] 10
